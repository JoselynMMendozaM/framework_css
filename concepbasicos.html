Conceptos básicos

Un conjunto de páginas web relacionadas y otros recursos digitales, generalmente identificados por un dominio común, que se encuentran en un servidor web y están disponibles en la World Wide Web.

Un documento digital accesible a través de Internet que puede contener texto, imágenes, videos, enlaces y otros elementos multimedia.
Las páginas web forman parte de un sitio web más amplio.

Un enlace interactivo que conecta una parte de un documento con otro documento, recurso o ubicación en la web.
Los hipervínculos permiten la navegación rápida entre páginas web y recursos relacionados.

Una dirección única que identifica la ubicación de un recurso en la web.
Una URL generalmente incluye el protocolo de transferencia (por ejemplo, "http" o "https"), el nombre de dominio y la ruta al recurso específico.

El lenguaje de marcado estándar utilizado para la creación y estructuración de contenido en la web.
Define la estructura básica de las páginas web mediante el uso de etiquetas y atributos.

Un lenguaje de estilo utilizado para controlar la presentación y el diseño de las páginas web.
CSS separa la estructura (definida por HTML) del estilo, permitiendo un diseño más consistente y flexible.

Un lenguaje de programación interpretado que se utiliza para agregar interactividad y dinamismo a las páginas web.
Permite realizar acciones del lado del cliente, como validar formularios, actualizar contenido dinámicamente y responder a eventos del usuario.

Un sistema que traduce los nombres de dominio legibles para los humanos en direcciones IP numéricas que las computadoras utilizan para identificarse entre sí en la red.

Un protocolo que asigna direcciones únicas a cada dispositivo conectado a una red, permitiendo la comunicación entre ellos.
En el contexto de la web, las direcciones IP identifican servidores y dispositivos en Internet.

Una identificación legible para los humanos que se asigna a una dirección IP para facilitar la navegación en la web. Los dominios web, como "ejemplo.com", se registran a través de registradores de dominios.

Un modelo de arquitectura de red en el que un cliente (por ejemplo, un navegador web) solicita recursos o servicios a un servidor (por ejemplo, un servidor web).
La comunicación entre ellos sigue el modelo solicitud-respuesta.

Un protocolo utilizado para la transferencia de archivos entre computadoras en una red. Se utiliza comúnmente para cargar archivos a un servidor web.

Un protocolo de comunicación utilizado para la transferencia de información en la World Wide Web.
HTTP define cómo se envían y reciben las solicitudes y respuestas entre clientes y servidores web.

Una versión segura de HTTP que utiliza cifrado SSL/TLS para proteger la comunicación entre el cliente y el servidor.
Se utiliza para garantizar la seguridad en la transferencia de datos sensibles.

Es aquello que ven los usuarios e incluye elementos visuales, como botones, casillas de verificación, gráficos y mensajes de texto. Permite a los usuarios interactuar con la aplicación.

Son los datos y la infraestructura que permiten que la aplicación funcione.

Se refiere al servicio que permite a individuos y organizaciones alojar sus sitios web y aplicaciones en servidores conectados a Internet

Es una opción intermedia entre el alojamiento compartido y el dedicado.

Es una forma de alojamiento web que utiliza recursos de múltiples servidores conectados en red.

Consorcio World Wide Web, es una organización internacional que se dedica al desarrollo de estándares para la World Wide Web.
Fue fundado por Tim Berners-Lee, el inventor de la web, en 1994, y está compuesto por un equipo diverso de expertos de diversas organizaciones alrededor del mundo.
